## kyawthuite
kyawthuite, or just kw - minimal and stable desktop OS based on Fedora, UBlue, Bazzite and CachyOS technologies.

kyawthuite (pronounced as "kyatuit")- named after the [Kyawthuite](https://wikipedia.org/wiki/Kyawthuite) mineral, a rare natural bismuth antimonate (BiSbO₄) known for its **stability and distinct crystalline structure**.

### system tools

to check for system updates (system image + flatpak packages + brew packages):
```bash
kw update
```

to upgrade system:
```bash
kw upgrade
```

or just use desktop files from application launcher: "Update system (only check)" and "Update system (perform)".

kyawthuite does not update automatically. do it yourself, in your preferred time and place.
